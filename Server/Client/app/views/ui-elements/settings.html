System Settings

Hands Free Leveler uses a clientless queue system to connect your smurfs, so this system requires packet structures to talk with the Riot servers. We have a special feature to detect new packets to make your smurfs secure. This process requires some time (3-5) minutes.

If you want to level up fast and spend some RP from your account, enable this option. When you are level 5, system will buy boost for you.

If your game is bugsplatting system could be outdated and it is dangerous to continue. But sometimes it is just a bugsplat, maybe everything is alright...

Disabling Gpu saves lots of rendering power from your computer and let you use your computer without seing any game windows. You can watch your game from Remote Controller anytime you want. (Uses a special method to disable drawing, does not replace config)

Sometimes Bol fails to inject to multiple games same time, use manual injection method if you want to fix this issue. Most of the AFK reports are because of this option is disabled
